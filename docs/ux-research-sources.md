# UX Research Sources

Curated list of trusted UX/UI blogs, publications, and resources. The UX Market Research agent uses this list as a starting point for research.

**Maintenance:** When the agent discovers a valuable new source during research, it will ask the user before adding it here.

---

- **Nielsen Norman Group** — https://www.nngroup.com — Foundational UX research and guidelines
- **Baymard Institute** — https://baymard.com — E-commerce and web UX research
- **Laws of UX** — https://lawsofux.com — UX principles with practical applications

- **Refactoring UI** — https://www.refactoringui.com — Practical visual design tips (by Tailwind creators)
